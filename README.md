# Sage implementation of the attack against the Oblivious PRF from supersingular isogenies

## Usage

1. `git clone https://github.com/isogenists/isogeny-OPRF.git`
1. `cd isogeny-OPRF/src/attack`
1. `sage one_more_attack.sage`
